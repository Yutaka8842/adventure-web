# アプリケーション名	
- Adventure

# アプリケーション概要
- 一緒に冒険する仲間を探すアプリケーション

# 主な機能
- ルームの一覧を見てルームに参加もしくはルームを作成することでチャットができる

# 実装予定の機能
- ルーム詳細編集機能
- スケジュール管理機能
- ToDo機能
- マイページ機能
- ギャラリー機能
